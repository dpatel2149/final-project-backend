# final-project-backend

## Project Overview
This project is the backend for a forum-style web application built using Node.js, Express, TypeScript, and MongoDB.

The backend supports user registration, login, post creation, retrieving posts, updating posts, deleting posts, commenting on posts, and logging incoming requests.

## Tech Stack
- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- JWT
- Bruno for API testing
